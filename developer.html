Задание для соискателей на должность программиста БД

Успешно выполнившие задание соискатели приглашаются на личное собеседование.

SQL

  1. В справочнике городов 2 поля — ID и NAME. ID — первичный ключ. Удалить дубликаты, оставить по одной записи с каждым названием города.
  2. Описать структуру базы данных предметной области «Расписание уроков».
  3. Написать процедуру, выводящую номер свободного кабинета для занятий в указанное время.

XSL

Исходный XML-документ содержит данные о всех выданных сотрудникам заданиях за год — сотрудник, дата, текст задания. 
<?xml version="1.0"?>
<rows>
  <row>
    <name>Иван Иванов</name>
    <date>01.01.2009</date>
    <text>Купить балалайку</text>
  </row>
  <row>
    <name>Пётр Петров</name>
    <date>05.01.2009</date>
    <text>Продать балалайку</text>
  </row>
...

1. Необходимо с помощью XSLT получить HTML документ, содержащий таблицу, в строках которой сотрудники, в столбцах — месяцы, на пересечении — количество заданий по сотруднику за месяц.
2. Необходимо с помощью XSLT получить HTML документ, содержащий по таблице на каждый месяц, в строках каждой таблицы сотрудники, в столбцах — дни месяца, на пересечении — количество заданий по сотруднику за день.

Ответ должен содержать:
  • Исходный файл XML с образцом входных данных, содержащий 100 заданий
  • Шаблоны XSLT, преобразующие исходный файл XML в результирующие HTML
  • Результирующие таблицы html
